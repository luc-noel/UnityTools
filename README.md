# Luc's Unity Tools
What it says on the tin. A dumping grounds for the various tools and utilities I made for Unity and want easy access to. Tested in Unity 2022.3.62f.

## BillboardObject
Simple billboard script with axis locks. Use the main camera if no target is provided.

<img width="491" height="197" alt="image" src="https://github.com/user-attachments/assets/2a3dfc0c-9a93-4dda-8596-02f06968b582" />

## EasyReplaceFile
Right-click menu to replace a selected file in Unity with any file. Can replace files of different types or differen names while keeping the original Unity file name.

## NoteKeeper
In-editor note organiser with customizable tags, themes, and sorting. Additional editor window for quick indexing and navigation to various note assets.

## SelectedObjectsCounter
Displays a count of the currently selected objects from the Hierarchy or Project at the bottom right corner of the Scene view. Toggleable using the C keybind or Tools menu item.
